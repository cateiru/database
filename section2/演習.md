# 演習1

## データベース shop を作成する

```sql
CREATE DATABASE shop;
```

## 商品テーブルを作成

```sql
CREATE TABLE 商品 (
    商品ID CHAR(4) NOT NULL,
    商品名 VARCHAR(100) NOT NULL,
    商品分類 VARCHAR(32) NOT NULL,
    販売単価 INTEGER,
    仕入単価 INTEGER,
    登録日 DATE,
    PRIMARY KEY (商品ID));
```
