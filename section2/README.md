## 検索の基本

### select

```sql
select 商品名, 商品分類 from 商品 where 商品分類='衣服';
```

- select句: 出力する列名を指定
- from句: 検索する表名を指定（省略可）
- where句: 検索する行の条件式を指定（省略可）


- 全列指定は`*`
- 列にその場限りの別名付与（`AS 別名`）
- 重複したデータの除去（`DISTINCT 列名`）
