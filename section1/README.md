## 使用Database

### PostgreSQL is 何？

- カリフォルニア大学バークレイ校のCS学科で開発された。
- オブジェクトリレーションデータベース管理システム。
- OSS
- 標準SQLの大部分をサポートしている。

### PostgreSQL の起動停止

```bash
# macOS の場合
# 起動
brew services start postgresql

# 停止
brew services stop postgresql

# 起動しているSQLの確認
brew services list
```
